# TagMyTaxi Platform Roadmap

## v2.1 — Q3 2026

- [ ] Multiple stops per ride (white-label configurable via `enabledFeatures.multipleStops`)
- [ ] Ride sharing / carpooling engine (matching service extension)
- [ ] Driver earnings dashboard with weekly/monthly analytics
- [ ] Passenger loyalty programme (points, tiers, rewards)
- [ ] Live in-app chat between passenger and driver (via LiveKit)

## v2.2 — Q4 2026

- [ ] Scheduled / pre-booked rides with driver assignment 30 min prior
- [ ] Corporate account management (expense codes, centralised billing)
- [ ] Dynamic pricing configuration UI for fleet managers
- [ ] Multi-language support expansion (Hindi, Urdu, French)
- [ ] iOS Widgets and Android dynamic shortcuts

## v3.0 — 2027

- [ ] Electric vehicle fleet management (range, charging stop routing)
- [ ] Autonomous vehicle dispatch integration (API-first)
- [ ] Advanced analytics platform (cohort analysis, LTV, churn prediction)
- [ ] Marketplace model (third-party driver applications)
- [ ] White-label mobile app skin builder (no-code branding)

## Completed

- [x] v2.0 — Monorepo restructure, microservices extraction, multi-tenant config layer
- [x] v1.x — Initial platform release
